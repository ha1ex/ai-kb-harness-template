# Саммари: интервью с CTO о пилоте AI-ассистента поддержки

> **Walkthrough-пример.** Слой `02_sources` — короткое саммари одного raw-артефакта,
> зеркалит его имя (`YYYY-MM-DD-…`). Именно на этот слой ссылаются synthesis/decisions.

Ключевые утверждения из интервью (2026-06-30):

- ~40% тикетов — повторяющиеся вопросы (интеграции, биллинг); средний тикет — 18 минут
  работы агента поддержки.
- Ограничение по данным: облачные API только с DPA; есть простаивающий GPU-сервер —
  локальный хостинг возможен.
- Критерий успеха пилота: deflection rate ≥ 20% на типовых категориях за первый месяц
  при CSAT ≥ 4.5; просадка CSAT — немедленная остановка пилота.
