# 13-day sprint with daily theme rotation

> Problem solved: Fixed-day-of-week sprint cadence routinizes work into a Groundhog Day loop; a 13-day rotation forces different task-types onto different weekdays sprint-to-sprint.

## What

A sprint pattern replaces the 14-day scrum cadence with a 13-day cycle, so the same task-theme never lands on the same weekday twice in a row. Each day of the rotation carries a theme (deep build / review / planning / customer / cleanup), and the off-by-one length keeps the rotation phase-shifting against the calendar week.
