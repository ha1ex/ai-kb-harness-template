# Calendar event tagging convention (#cohort-02)

> "Which meeting did we discuss X in?" Five seconds tagging events with project IDs makes every transcript filterable three months later.

## What

Tag every calendar event with a cohort/project identifier; downstream transcripts saved by the meeting agent inherit the tag automatically → all retrievals become filterable by cohort, project, or initiative. Five seconds when scheduling, infinite ROI when searching three months later.
