# Bootstrap personal context from Telegram + Notion export

> New vault is empty on Day 1 — agent has nothing to work with. Export Telegram + Notion + ChatGPT/Gemini history; ingestion seeds it in one hour.

## What

Telegram Lite → Export Chat History → ~30K messages across 5 HTML files. Notion → bulk export. One prompt: "Here's my context. Build a vault using {project} {type} – YYYY-MM-DD.md naming. Find patterns, create folders, generate context cards per person." Bridge across sources via Telegram username (unique cross-source key). ~1 hr from zero to fully context-aware vault.
