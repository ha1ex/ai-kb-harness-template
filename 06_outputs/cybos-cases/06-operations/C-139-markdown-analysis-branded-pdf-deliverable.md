# Markdown analysis → branded PDF deliverable

> Problem solved: AI contract-review (and other analysis) outputs ship as bare Markdown; consultants and ops teams need a polished PDF that sells the work product to a non-technical signatory.

## What

A PDF-rendering pattern takes a Markdown analysis and emits a branded report via ReportLab — cover page, a speedometer-gauge risk score, and a risk-dashboard layout with severity color bands. The reusable pattern: render any structured analysis to a client-grade PDF deliverable rather than handing over raw Markdown.
