# AI adoption baseline + target measurement framework

> "Are we more AI-native than last quarter?" gets vibes for an answer. Four metrics per team (adoption %, heavy %, workflows, hours saved) replace vibes.

## What

Per-team baselines + targets across four metrics: overall adoption % (e.g., 16% → 80%), heavy-user % (6% → 30%), production workflows (8 → 80+), hours saved/month. Depth distribution buckets (non-user / light / medium / heavy). Before/after proof points anchored in real conversions (Oracle→Postgres 6mo → <1mo; 600h manual → 3h; release notes 2–3h → 10 min).
