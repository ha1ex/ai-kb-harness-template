# 7-process self-rating map — founder's lightweight AI-maturity diagnostic

> Full Diagnostic-to-Dashboard takes 21 interviews. 30 minutes: rate 7 processes 1-10; mark a Champion per process; flag the ones with no Champion.

## What

Founder lists 5-10 core processes (Sales / Marketing / Finance / Tech-support / Onboarding / Admin / Design / HR) → self-rates each 1-10 on current AI-adoption maturity → marks the Champion (named person) per process → flags processes with no Champion as founder-bottlenecks. Take 30 minutes; redo monthly. Cheaper than a full diagnostic; surfaces the same prioritization.
