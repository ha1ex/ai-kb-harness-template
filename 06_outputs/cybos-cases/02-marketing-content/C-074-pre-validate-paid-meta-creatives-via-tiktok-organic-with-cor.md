# Pre-validate paid Meta creatives via TikTok organic — with correlation caveat

> Problem solved: burning Meta paid budget on creative concepts you could have killed for free.

## What

Pattern: post candidate videos organically to TikTok; promote only the ones that go viral organically into paid Meta. Free A/B test in a different feed. Caveat surfaced by other operators: TikTok virality correlates weakly with Meta ranking — different algorithm weights, different audience intent — so use TikTok organic as a directional filter, not a determinative gate. Best fit for top-of-funnel hook testing.
