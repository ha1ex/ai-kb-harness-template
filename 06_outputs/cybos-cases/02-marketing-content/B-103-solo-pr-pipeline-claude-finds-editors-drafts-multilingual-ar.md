# Solo PR pipeline — Claude finds editors, drafts multilingual articles, sends them

> Problem solved: Indie founders and hobby-project owners can't afford a PR agency; Claude with email access can run targeted-region placement end to end and land real published articles.

## What

Plug Claude Code into the founder's email account, brief it on the project, and ask it to find editor/author contacts at outlets in target regions, draft personalised articles in N languages, and send each. The founder edits lightly before send. Real placements are achievable — one operator landed a published article in regional press from a single multi-language outreach run.

## Why it matters

Early-stage and hobby projects rarely justify a PR retainer ($3k–10k/mo). This collapses placement to a few hours of founder time. The unique unlock is multilingual drafting: editors at non-English outlets respond to articles already written in their language at a much higher rate than to English pitches.

## End-to-end

1. Connect Claude Code to the founder's email account (Gmail MCP, IMAP/SMTP, or equivalent).
2. Write a tight brief: project context, target regions, target outlet types (general press / niche industry / lifestyle), desired tone.
3. Ask Claude to find editor and author contacts at relevant outlets in the target regions. Validate names against public bylines.
4. Ask Claude to draft a personalised article version per outlet — three languages if the regions are multilingual (one operator used original-language + English + a regional language).
5. Founder reviews and lightly edits each draft.
6. Claude sends each personalised email with the draft attached or pasted inline.
7. Track replies; respond to interested editors with revisions.

## Gotchas

## Generic English pitches sent to regional press get ignored. The multilingual personalisation is the unlock — and Claude has to be told the target language per outlet, not left to guess from the domain TLD. Also: do not let the agent send without human review on the first batch; a hallucinated stat in a pitch email burns the relationship permanently.

## Tools

- Claude Code with email access (Gmail MCP, IMAP, or local mail client integration)
- A project brief document
- Verified editor contact lists (Claude finds, founder confirms)
