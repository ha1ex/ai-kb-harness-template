# Harmonic saved-search → rule filter → Sonnet screener → Opus L3 pipeline

> Twitter-dependent sourcing breaks when accounts get banned. Saved-search → rule filter → cheap screener → Opus only on survivors.

## What

Independent sourcing pipeline with no Twitter dependency: poll four Harmonic saved searches → deterministic rule filter (founder/CEO/CTO title + ≥2 prior roles for persons; vertical keyword + <500 headcount for companies) → Sonnet RESEARCH/SKIP screener → Opus deep evaluation. Two-stage funnel keeps Opus calls rare.
