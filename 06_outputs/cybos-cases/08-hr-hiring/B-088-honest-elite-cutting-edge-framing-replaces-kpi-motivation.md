# Honest "elite / cutting-edge" framing replaces KPI motivation

> Bonus programs and KPI trees engineer compliance, not commitment. Honest "we're at the frontier; run with us or part on good terms" hits <10% refusal.

## What

Replace the standard "AI bonus program + champion network + KPI tree" with an honest, one-paragraph framing: "What we're doing right now is at the world-class frontier. Few companies in the world are doing this. If you want to run with us, let's run. If you're not ready, let's part on good terms with severance — no shame." That framing does 80 %+ of the motivational lifting; the rest is doing the work in public.

## Why it matters

One operator reported < 10 % direct refusal rate across 15–17 departments after switching from KPI-based motivation to this framing. The bonus program approach (A-015) had been tried multiple times and consistently failed at the margin; the elite-framing approach didn't try to engineer motivation, it simply honest-priced the choice.

## End-to-end

1. Draft the message in one paragraph. Three elements: this is rare in the world; we're doing it; opt-in or opt-out with dignity.
2. Deliver in person, all-hands, founder-voice. Not Slack. Not email-first.
3. Set the severance offer concretely (X months pay, no NDAs, no shame). The offer must be real, otherwise the framing is hollow.
4. Open Q&A — answer every question, including "what if I can't learn this fast enough?"
5. Make a 2-week opt-out window. Anyone leaves in 2 weeks with severance.
6. Reinforce weekly: show visible wins. "Look what we shipped this week" is the only motivation that compounds.
7. Don't add KPIs in month 3. The framing only works if it stays clean.

## Gotchas

- Don't ship this framing if you haven't done the work yourself first. Operators who say "we're at the frontier" while still asking PMs to write tickets manually get exactly the cynicism they earn.

See also: C-064 (no-metrics counter-stance), C-066 (no-token-optimization counter-stance).

## Tools

- Founder credibility (genuinely worked with AI yourself before saying this)
- A real severance budget
- Patience for the 2-week window
