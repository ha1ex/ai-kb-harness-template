# Cowork tab in Claude desktop app (non-CLI onramp)

> Non-developers won't open a terminal. The Claude desktop app's Cowork tab points at a folder and runs the same prompts.

## What

The Claude desktop app's Cowork tab is a friendlier interface than raw terminal Claude Code — point it at a folder (notes / Dropbox) and issue the same prompts. Lower barrier for non-developers; useful for spreading AI-native workflows beyond engineering.
