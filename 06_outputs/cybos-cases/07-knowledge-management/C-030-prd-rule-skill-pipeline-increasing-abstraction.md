# PRD → Rule → Skill → Pipeline (increasing abstraction)

> Every shipped artifact teaches the same lesson the next one. Distil each finish into a rule file; the next agent doesn't repeat dead ends.

## What

A methodology ladder: PRD = pre-build requirements; Rule = post-build distillation of "what worked, how to repeat it"; Skill = packaged rule with prompts; Pipeline = multi-step skill. After every shipped artifact, distill the next-rung rule file so the next agent doesn't repeat the same dead ends.
